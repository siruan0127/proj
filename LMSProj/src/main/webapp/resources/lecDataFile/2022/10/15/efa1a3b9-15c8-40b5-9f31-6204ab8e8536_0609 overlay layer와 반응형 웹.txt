CSS 추가규칙

스타일시트에 css가 아닌데도 들어가는 속성이 있음
@가 붙음

@import

@font-face
사용자의 컴퓨터에 설치되지 않은 글꼴을 사용하도록 설정
웹 폰트를 사용자의 브라우저에서 다운로드 해 사용하도록 지정
용량 등의 문제로 많이 사용하지 않는 규칙

@media(장치)
다양한 장치에서 HTML 문서가 적절한 형태를 갖도록하는 규칙
하나의 HTML 문서로 다양한 장치에서 사용하기 위해 부각되는 규칙
@media 규칙은 미디어 타입과 함께 사용
->링크 요소에 속성값으로 들어갈 수 있음
<link rel=""  href=""  media= "screen">
<link rel=""  href=""  media= "print">


미디어 타입을 통해 출력하는 기기를 구분
all(모든 미디어 타입)
*screen(화면에 출력)
print(프린터로 출력)

미디어 규칙을 사용해서 작성해 놓은 블럭을 미디어 쿼리라고 부른다.

@media screen {
	html, body { height:100%;}
	body {color: white;}
}

@media print (이 안에 지정한 스타일 속성은 출력할 때만 보임)
